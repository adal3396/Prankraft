# Prankraft Portfolio — The Trickster Site 🎭

> A premium developer portfolio that **looks** flawless but **behaves** mischievously.

## 🎯 Concept

This is a sleek, modern developer portfolio website designed for the **Prankraft** front-end web design competition (IEEE Computer Society). At first glance, it appears to be an ultra-polished professional portfolio. But every interaction hides a playful surprise — buttons flee from your cursor, the page flips upside down, a fake Blue Screen of Death appears, and much more.

**The deception:** It *looks* incredibly professional but *behaves* chaotically and playfully.

## 🎭 Prank Elements (Spoilers!)

| Interaction | What Happens |
|---|---|
| **Hover nav links** repeatedly | Text scrambles into funny alternatives ("Skills" → "🎭 Magic") |
| **Click "Hire Me"** 5 times | Begs you progressively, then celebrates with confetti |
| **Click "View My Work"** twice | Page flips upside down briefly |
| **Click "Download CV"** | Fake download → 404 → CV button runs away |
| **Hover skill cards** | Cards flee from your mouse cursor |
| **Click testimonials** | Reviews swap to absurd quotes (from "A Bug", "Coffee Machine", etc.) |
| **Submit contact form** 1st time | Triggers a fake Windows BSOD |
| **Submit form** 4th time | Gravity mode — everything falls down |
| **Click the logo** multiple times | Colors invert, go grayscale, then party mode |
| **Click scroll indicator** | Scrolls the WRONG direction |
| **Click about photo** | Avatar emoji keeps changing (identity crisis) |
| **Social media links** | Random excuses ("My social life is also 404") |
| **Konami Code** (↑↑↓↓←→←→BA) | Rainbow mode + everything floats |
| **Scrolling** | Page randomly "slips" sideways |

## 🛠️ Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, gradients, glassmorphism, animations
- **Vanilla JavaScript** — No frameworks, pure JS for all interactions
- **Google Fonts** — Outfit, Inter, JetBrains Mono

## 🚀 Deployment

### Deploy to Vercel
1. Push to GitHub
2. Import in Vercel
3. Framework Preset: **Other**
4. Output directory: `.` (root)
5. Build command: leave empty or `echo done`

### Deploy to GitHub Pages
1. Push to GitHub
2. Settings → Pages → Deploy from main branch

## 💡 Design Philosophy

- **First impression**: Premium, dark theme, smooth animations — a "serious" portfolio
- **Second impression**: Chaos. Controlled, delightful chaos.
- **All pranks are**: Non-harmful, user-safe, and fun
- **Responsive**: Works on both mobile and desktop
- **No external dependencies**: Pure HTML/CSS/JS
