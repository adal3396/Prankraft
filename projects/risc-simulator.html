RISC CPU Simulator

A full-featured RISC processor simulator with visual step-by-step execution, register inspection, and memory visualization.

🚀 Live Demo 📦 Source Code

Overview

The RISC CPU Simulator provides a complete educational environment for understanding processor architecture. Users can write assembly programs, step through execution cycle-by-cycle, and observe how each instruction modifies registers, memory, and the program counter in real-time.

Tech Stack

TypeScript React Express.js Node.js MongoDB Monorepo Architecture Vercel

Key Features

⚡

Step-by-Step Execution

Watch each instruction execute cycle-by-cycle with highlighted pipeline stages

📝

Assembly Editor

Built-in code editor with syntax highlighting and error detection

🔍

Register Inspector

Real-time view of all 32 registers with change highlighting

💾

Memory Viewer

Hex/decimal memory visualization with read/write tracking

Challenges & Learnings

  • Monorepo management: Coordinating TypeScript types across core simulation engine, Express API, and React frontend required careful workspace configuration.
  • Instruction pipelining: Accurately modeling pipeline hazards and forwarding logic was the most complex algorithmic challenge.
  • Vercel deployment: Migrating the Express backend to Vercel Serverless Functions required restructuring the API layer while maintaining the monorepo structure.
