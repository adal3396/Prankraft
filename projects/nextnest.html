NextNest Platform

A community platform connecting donors with orphanages, featuring real-time inventory tracking, multi-role portals, and donation management.

🚀 Live Demo 📦 Source Code

Overview

NextNest bridges the gap between donors who want to help and orphanages that need support. The platform provides dedicated portals for donors, orphanage administrators, and super admins, enabling transparent donation tracking, inventory management, and communication.

Tech Stack

Next.js 14 Supabase Tailwind CSS TypeScript PostgreSQL Row Level Security Vercel

Key Features

👥

Multi-Role Portals

Separate interfaces for donors, orphanage admins, and super administrators

📦

Inventory Tracking

Real-time tracking of donated goods with status updates and history

🔐

Row Level Security

Supabase RLS policies ensure data isolation between organizations

📊

Admin Dashboard

Comprehensive analytics for donation trends, inventory levels, and user activity

Challenges & Learnings

  • Multi-tenant security: Implementing Row Level Security policies in Supabase that correctly isolate orphanage data while allowing donors cross-organization visibility.
  • Real-time updates: Using Supabase Realtime subscriptions to instantly reflect donation status changes across all connected clients.
  • Seed data generation: Creating realistic test data with proper relationships between donors, orphanages, and donations required careful orchestration.
