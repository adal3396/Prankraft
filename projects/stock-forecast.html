Stock Forecast AI

CNN-based stock price prediction using Short-Time Fourier Transform analysis with real-time visualization dashboard.

🚀 Live Demo 📦 Source Code

Overview

Stock Forecast AI transforms raw stock price data into frequency-domain representations using Short-Time Fourier Transform (STFT), then applies Convolutional Neural Networks to predict future price movements. The system features a full web dashboard for visualizing predictions, historical data, and model performance metrics.

Tech Stack

Python TensorFlow / Keras React Node.js STFT Analysis CNN Architecture Pandas / NumPy Chart.js

Key Features

📊

STFT Preprocessing

Converts time-series stock data into spectrogram images for CNN processing

🧠

CNN Prediction Model

Multi-layer CNN trained on frequency features to predict price movements

📉

Real-time Dashboard

Interactive charts showing live predictions vs actual prices

📈

Model Evaluation

Comprehensive metrics: MSE, MAE, R² score, and confusion matrices

Challenges & Learnings

  • Overfitting prevention: Implemented dropout layers and data augmentation to prevent the model from memorizing training data patterns.
  • STFT window sizing: Finding the optimal window size required extensive experimentation — too small loses frequency resolution, too large loses temporal resolution.
  • Real-time updates: Balancing prediction accuracy with response time for the live dashboard required efficient caching and batch processing.
